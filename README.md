NHS England A&E Performance Analysis — March 2025
SQL and Tableau analysis of 4-hour compliance across all 187 NHS trusts in England using publicly available NHS England data.
Key Findings

National compliance: 73.8% against a 95% target
91 trusts (49% of England) classified RED — critically failing
North West worst-performing region at 76.1% average compliance
116 trusts recorded patients waiting over 12 hours
Shrewsbury and Telford highest risk at 9.94% 12-hour wait rate

Tools
SQL (SQLite) · Tableau · Excel
